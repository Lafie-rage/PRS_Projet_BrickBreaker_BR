Detailed Description

A list of clients.

Containts the lists of the clients the server knows or want to communicate to as array of t_client. Also contains is size. The clients undefined have the PID -1. Those clients are in the client's array from size to the length of the array. Avoid browsing the array with anything else than a for loop from 0 to size-1.
